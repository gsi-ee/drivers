This folder collects some (deprecated) tools for GOSIP with PEXOR/KINPEX PCIe optical receiver.
This version was ported and tested on  Debian9 Linux (kernel 4.9.30, gcc 6.3.0) by JAM (j.adamczewski@gsi.de).


IMPORTANT NOTE: For Linux X86 Systems, it is RECOMMENDED to work with mbspex library and gosipcmd tool instead!
This is available at https://subversion.gsi.de/dabc/drivers/mbspex
